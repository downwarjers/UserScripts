# 巴哈姆特動畫瘋資訊+ (BahaAnimeInfoMod)

這是一個專為 [巴哈姆特動畫瘋](https://ani.gamer.com.tw/) 設計的 UserScript 。
它會自動分析當前觀看的動畫，並從外部資料庫抓取詳細的 **CAST (聲優)** 與 **主題曲 (OP/ED)** 資訊，直接顯示在影片播放頁面下方，補足官方資訊的缺漏。

## ✨ 主要功能

* **自動獲取詳細資訊**
    * **CAST 列表**：列出角色名稱及其對應的聲優 (CV)。
    * **主題曲資訊**：列出 OP (片頭曲)、ED (片尾曲) 及插入曲的歌名與歌手。

* **智慧分頁系統 (多季度/系列支援)**
    * 自動偵測該動畫的相關系列作（例如：第一季、第二季、外傳、劇場版）。
    * 以 **分頁 (Tabs)** 形式整合顯示，讓您在同一個頁面就能快速切換查看不同季度的製作資訊，無需跳轉頁面。

* **便捷的外部連結**
    * **Wiki 百科**：點擊聲優名字可直接連結至 Wikipedia。腳本會自動優先搜尋**中文維基**，若無則連結至日文維基。
    * **YouTube 搜尋**：點擊主題曲資訊，可直接在 YouTube 上搜尋該歌曲，方便試聽。

* **雙重資料來源 (高覆蓋率)**
    * 主要來源：優先查詢 [Syoboi (しょぼいカレンダー)](https://cal.syoboi.jp/)，提供最準確的日本動畫播出資訊。
    * 備用來源：若 Syoboi 查無資料，會自動切換至 [allcinema](https://www.allcinema.net/) 進行查詢，確保冷門番也能找到資料。

* **自動/手動模式切換**
    * 預設為自動載入資訊。
    * 可透過腳本選單切換為「手動模式」，在需要時點擊按鈕才讀取資料，節省資源。

## 🚀 安裝方式

1.  請先安裝瀏覽器擴充功能：[Tampermonkey](https://www.tampermonkey.net/) (Chrome/Edge/Firefox)。
2.  點擊安裝腳本：**[BahaAnimeInfoMod.user.js](https://github.com/downwarjers/userscripts/raw/main/BahaAnimeInfoMod/BahaAnimeInfoMod.user.js)** (請確認連結指向您的 Raw 檔案位置)。
3.  安裝完成後，重新整理動畫瘋的影片頁面即可生效。

## 📖 使用說明

1.  進入任一 **巴哈姆特動畫瘋** 的影片播放頁面。
2.  將網頁向下捲動至「作品資料」區塊。
3.  腳本會自動在該區塊內插入一個 **「aniInfo+」** 的欄位，並開始載入資料。
4.  若該作品有多個季度（如 Syoboi 資料庫中有關聯），上方會出現分頁標籤，點擊即可切換查看。

## ⚠️ 注意事項

* 本腳本依賴外部網站 (Syoboi, allcinema, Wikipedia) 的 API 或頁面結構，若來源網站改版可能會導致功能暫時失效。
